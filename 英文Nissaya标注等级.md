# 英文Nissaya标注


√: 支持
×: 不支持
○: 部分支持

| Items | Basic | Advance | 
| - | - | - |
| 词意 | √ | √ |
| 格位 | √ | √ | 
| 超长单词连字符 |  √ | √ |
| Sandhi拆开形式| √ | √ |
| Sandhi连字符| √ | √ |
| 单词用法 | ○ |  √ |
| 关系连线 | × |  √ |
| 词序调整 |  ○ | ○ |
| 增补单词 |  ○ | ○ |

- `Basic` 第一阶段需要支持
- `Advance` 在使用者要求时可以支持
- `词序调整` `增补单词` 依照现有的缅文nissaya，视不同句子情况而定。
